# Math Game Console App

Welcome to the **Math Game Console App**! This is a simple math game made in C# to practice basic operations and have fun while learning.

## Features
- Four math operations: Addition, Subtraction, Multiplication, and Division
- Three difficulty levels: Easy, Medium, and Hard
- User answer history
- Friendly console interface

## How to Play
1. Run the program.
2. Choose one of the menu options:
   - `1` - Addition
   - `2` - Subtraction
   - `3` - Multiplication
   - `4` - Division
   - `9` - View answer history
   - `0` - Exit
3. Choose the desired difficulty:
   - `1` - Easy
   - `2` - Medium
   - `3` - Hard
4. Answer the 10 questions presented.
5. See your score at the end of each round.

## How to Run
1. Make sure you have the [.NET 9.0 SDK](https://dotnet.microsoft.com/download) installed.
2. In the terminal, navigate to the project folder.
3. Run:
   ```powershell
   dotnet run
   ```

## Project Structure
- `Program.cs`: Main game code
- `CodeReviews.Console.MathGame.csproj`: Project configuration file
